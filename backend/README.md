# DevCamper API

> Backend API for DevCamper application, which is a bootcamp directory website

## Projects Structures



## Usage
 
Rename "config/config.env.env" to "config/config/config.env" and update the values/settings to your own

## Install Dependencies
```
npm install
```

## Run App 
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
